# Hex Editor

A cross-platform advanced hex editor and malware analysis tool for Windows and Linux, built with Python and Tkinter.

## Features
- Hex/ASCII editing with change highlighting
- PE/ELF executable analysis
- Malware scanner (40+ indicators)
- Auto assembler/disassembler (x86/x64)
- Metadata editor
- Encoding support (20+ formats)
- Linux compatibility
- Error checking and auto-fix for PE/ELF files
- Context menu for error fixing

## Usage
1. Install Python 3.7+
2. Run `HexEditor.py`:
   ```
   python HexEditor.py
   ```
3. Open any file to view/edit in hex, scan for malware, analyze structure, and more.

## Requirements
- Python 3.7 or newer
- No external dependencies (uses only standard library)
